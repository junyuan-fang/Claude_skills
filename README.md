# Claude Skills

Personal Claude Code skills backup.

## Skills

- **confluence** - Confluence CLI skill for reading, searching, creating and updating Confluence pages (`@qunhe/confluence-cli`)

## Setup

```bash
# Clone to Claude skills directory
git clone git@github.com:junyuan-fang/Claude_skills.git ~/.claude/skills
```

## Add new skills

```bash
cd ~/.claude/skills
git add .
git commit -m "Add <skill-name> skill"
git push
```
