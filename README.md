# Claude Skills

Personal Claude Code skills backup.

## Skills

| Skill | Description |
|-------|-------------|
| **confluence** | Generic Confluence CLI skill — read, search, create, update pages via `@qunhe/confluence-cli` |
| **cf-qunhe** | Internal Confluence (cf.qunhequnhe.com) workflows — MCP-based read/edit, mermaid/markdown macro handling, page-to-markdown conversion, robot-mobot permission tips |

## Setup

```bash
# Clone to Claude skills directory
git clone git@github.com:junyuan-fang/Claude_skills.git ~/.claude/skills
```

## Sync changes

```bash
cd ~/.claude/skills
git add .
git commit -m "Update skills"
git push
```
